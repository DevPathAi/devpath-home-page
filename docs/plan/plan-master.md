## Vision

### 10x Check
홈페이지는 브로슈어가 아니라 살아있는 증명이다. 방문자가 도착한 순간 가입 없이 20초 안에 핵심 마법을 직접 겪는다. 심사위원은 주장을 읽지 않고 LCS가 작동하는 걸 눈앞에서 본다. 학습자는 "내 얘기네" 하고 진단을 시작한다. 실제 traction 숫자가 오르고, 창업자의 "2년제→학원→독학→그래서 만든다" 서사가 스크롤로 펼쳐진다. 1초 로딩, 예쁜 공유, 한/영 대응.

### Platonic Ideal
심사든 학습자든 60초 안에 떠나면서 셋을 안다 — DevPath가 뭘 하는지(느꼈으니까), 왜 다른지(LCS가 작동하는 걸 봤으니까), 왜 이 창업자를 믿는지(서사가 꽂혔으니까). 한 픽셀도 낭비 없이 믿음을 클릭(진단 시작/미팅 요청)으로 전환.

## CEO 우려 (기록, mode는 사용자가 EXPANSION 선택)
- **알파와의 시간 경쟁:** 본선을 이기는 건 홈페이지가 아니라 동작하는 알파 데모(장기전략 문서 본인 진술). 위 확장 4종은 총 ~7일 인간공수. 알파 스프린트 일정과 충돌하지 않게 타임박스/우선순위 관리 필요.
- **완화책:** 확장 위젯은 전부 캐닝/정적 구현 가능(실 알파 로직 의존 X). traction 티커만 실데이터 연결 — 숫자 하한 임계값으로 "빈약해 보임" 방지.

## Review Fixes (2026-07-13 CEO 심층 리뷰 반영)
- **F1 traction 빈/실패 상태:** 숫자가 하한(예: 가입 20) 미만이거나 fetch 실패 시, 티커를 "베타 진행 중 — 초대받기" 정적 메시지로 대체. 깨짐/빈약 모두 방지. 엔지니어링 기본값 명시.
- **F2 점진적 향상 강제:** 핵심 콘텐츠+주 CTA+본문+SEO/OG는 서버 정적으로 즉시 렌더·크롤링. 위젯 JS 4종은 해당 섹션 진입 시 lazy-load. JS 꺼짐 상태에서도 페이지 본질(정보+CTA+서사 텍스트)은 작동.
- **F3 공개 traction 엔드포인트 안전:** 정적 사이트라 인증 없음 → Apps Script read 엔드포인트는 **집계 카운트만** 반환, 원시 리드/PII 절대 노출 금지. 기존 리드폼 프라이버시 가드(소스코드·토큰·로그 차단) 통합 폼에 보존.

## Eng Review (2026-07-13)
**Stack 확정:** 바닐라 ES모듈 + Vitest(유닛) + Playwright(E2E 1). [Layer 1] 프레임워크·번들러·애니메이션 라이브러리 없음. lazy-load=IntersectionObserver+dynamic import, 모션=CSS transition + prefers-reduced-motion.

**구현 반영 발견:**
- 단일 `api.js` fetch 클라이언트(timeout·에러 정규화·재시도) — DRY, 5벌 중복 방지
- Apps Script 단일 Web App `?action=lead|stats`, `stats`는 화이트리스트 집계 필드만(F3)
- CORS: 라이브 Pages 도메인에서 Apps Script 호출 검증(로컬만 통과하는 함정)
- traction: localStorage 5분 TTL 캐시 + F1 임계/실패 fallback
- 위젯 lazy-load(IntersectionObserver + dynamic import), 이미지 width/height 명시(CLS 방지)
- **최우선 테스트:** traction F1 3분기(임계 미만/실패/정상) 유닛 — 심사-facing 회귀 방어선 + Playwright E2E 1

## Design Spec (2026-07-13 design review, 5/10 → 9/10)
분류: MARKETING/LANDING. DESIGN.md는 토큰만 상속(표현은 자유). 목업: 미생성(OpenAI 키 필요 — `$D setup` 후 재생성 가능).

**토큰 (DESIGN.md 정렬):**
- 컬러: 브랜드/텍스트 인디고-600 `#4f46e5`, CTA 면 인디고-500 `#6366f1`(텍스트 금지), bg slate-50 `#f8fafc`, 본문 slate-900 `#0f172a`, 보조 slate-500 `#64748b`. 전부 CSS 변수.
- 폰트: 본문/UI **Pretendard**, LCS 코드 답변 **D2Coding**. system-ui/Roboto 금지. 한글 행간 본문 1.6.
- 8pt 그리드, 버튼 라운드 8, 장식 그림자 금지(hairline slate-200 보더).

**Hero (결정: 에디토리얼 + 인라인 미니진단):**
- slate-50 솔리드 배경(**인디고 그라디언트 배경 금지 — AI-slop #1**). 인디고는 워드마크·주 CTA·강조 텍스트에만.
- Hero 예산: 워드마크 + 헤드라인 1("내 수준에 맞는 다음 단계를 AI가 안내") + 지지문장 1 + 주 CTA "내 실력 진단받기" + **인라인 미니진단 위젯 1**. 카드 금지.
- 좌: 브랜드/헤드라인/CTA · 우: 미니진단(바로 체험).

**기능/차별화 (결정: LCS 앵커 + 비대칭):**
- LCS 나란히 데모 = 전폭 시각 앵커. 진단/로드맵/AI멘토 = 비대칭 보조 블록. **3열 균등 카드 금지 — AI-slop #2.** 섹션당 한 가지 일.

**위젯 상태표:**
```
위젯       | LOADING     | EMPTY/저조                | ERROR         | SUCCESS
미니진단   | 스켈레톤3   | (캐닝)                    | "다시" 인라인 | 샘플 로드맵
LCS데모    | 즉시(캐닝)  | —                         | 정적예시 유지 | 나란히 비교
traction   | 스켈레톤    | "베타 진행 중—초대받기"(F1)| 동일 fallback | 숫자+aria-live
리드폼     | 버튼스피너  | —                         | 필드별 메시지 | "확인 메일 발송"
```
빈/저조 = 따뜻한 카피 + 단일 1차 행동("결과 없음" 금지).

**여정 아크:** 5초(내 얘기네)→5분(진단 체험)→관계(창업자 신뢰). 심사·유저 두 아크 병기.

**반응형/a11y:** 뷰포트별 의도(모바일=스택 금지). 대비 ≥4.5:1(인디고=primaryText), 터치 ≥44px, 포커스 링, `lang="ko"`, **traction 갱신 `aria-live="polite"`**, 색+레이블 병행, prefers-reduced-motion(스크롤리텔링 정적 폴백).

**AI-slop 회피 규칙:** 중앙정렬 남발·이모지 장식(→Material Symbols)·좌측 컬러보더 카드·"Welcome to/Unlock" 카피·쿠키커터 섹션 리듬 금지.
